wappsdev Tofu/Terraform Provider Registry

This is a private Terraform Provider Registry hosted by wappsdev.

Available providers

  • wappsdev/coolify — Coolify v4 provider (forked from SierraJC/terraform-provider-coolify, adds coolify_application resource via PR #87)

Usage

terraform {
  required_providers {
    coolify = {
      source  = "registry.wapps.co/wappsdev/coolify"
      version = "~> 1.0"
    }
  }
}

Trust

Releases are GPG-signed with the wappsdev-sigs key (key ID 6EB7E199D92EA30D, ED25519, expires 2028-05-27).
